# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this SDK, please report it
privately using GitHub's **[Report a vulnerability](https://github.com/highnote-oss/nodejs-sdk/security/advisories/new)**
form (also reachable via the **Security** tab → **Report a vulnerability**).

Please do **not** disclose the issue publicly until we have had a chance to
investigate and respond.

When reporting, include where possible:

- A description of the vulnerability and its impact
- Steps to reproduce, or a minimal proof-of-concept
- The SDK version(s) affected
- Any suggested mitigations

We will acknowledge receipt within 3 business days and work with you on a
disclosure timeline.

## Supported Versions

Only the latest published version of `@highnote-oss/nodejs-sdk` receives
security updates. Please upgrade to the latest version before reporting.
